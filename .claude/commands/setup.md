このプロジェクトの開発環境セットアップを行います。以下の手順を順に実行し、各ステップの結果を簡潔に報告してください。

## 1. Node.js の確認

`node -v` を実行し、バージョンを確認してください。
- 推奨バージョンは **Node.js 24.11.1** です。
- メジャーバージョンが異なる場合はその旨をユーザーに伝えてください。ただしセットアップは続行してください。

## 2. pnpm の確認

`pnpm -v` を実行してください。
- コマンドが存在する場合はバージョンを報告してください。
- 見つからない場合は以下のいずれかを案内し、ユーザーに選択を求めてセットアップを一時停止してください。
  ```bash
  # 推奨: Corepack を使う方法
  corepack enable && corepack prepare pnpm@latest --activate

  # または npm でグローバルインストール
  npm install -g pnpm
  ```

## 3. 依存関係のインストール

`pnpm install` を実行してください。完了したらパッケージ数を報告してください。

## 4. 環境変数ファイルの確認

`.env.local` が存在するか確認してください。
- 存在しない場合は `.env.example` の内容を確認し、ユーザーに以下を案内してください。
  ```bash
  cp .env.example .env.local
  ```
  コピー後、`BETTER_AUTH_SECRET`（`openssl rand -base64 32` で生成）と Google OAuth の `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` の設定が必要な旨を伝えてください。

## 5. 完了報告

全ステップが完了したら次のステップをユーザーに案内してください。

```
次のステップ:
1. /db-setup     — データベースのセットアップ（初回は必須）
2. pnpm dev      — 開発サーバーの起動
```
